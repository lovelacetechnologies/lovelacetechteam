DWM Hotkeys

  • Super+p --> Choose a program to launch via dmenu
  • Super+Return --> Open a new kitty instance
  • Printscreen --> Take a screenshot using flameshot
  • Super+PrintScreen --> Take a fullshot using flameshot
  • Super+F2 --> Open Firefox
  • super+F11 --> Launch Slock
  • super+q --> Kill active window
  • Super+Left_Arrow --> Move focus down the stack
  • Super_Right_Arrow --> Move focus up the stack
  • Super+comma --> Move focus to the next monitor
  • Super+period --> Move focus to prior monitor
  • Super+Shift+comma --> Move focused window to the next monitor
  • Super+Shift+period --> Move focused window to the previous monitor
  • Super+Shift+p --> Open power off options
  • Super+[ --> Increament gaps between windows
  • super+] --> Decrement gaps between windows
  • Super+equal --> increment number of master windows
  • super+minus --> Decrement number of master windows
  • super+b --> Toggle status bar on or off.
  • super+up_arrow --> Pulse mixer up 5%
  • super+down_arrow --> Pulse mixer down 5%
  • super+n --> Toggle night mode on and off
  • super+Shift+c --> Select new item out of clipboard
